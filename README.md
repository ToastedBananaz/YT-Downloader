# How to use

<details>
  <summary><h3>Setting up (Windows Only)</h3></summary>
  
1. Download the ZIP from Github
2. Right click the ZIP file then **Extract All**
3. Open the folder
4. Run **setup.bat** to install some stuff
5. if you want delete **setup.bat**
</details>

<details>
  <summary><h3>Download</h3></summary>

1. Run **start.bat**
2. Enter the YouTube URL
3. give a name
4. The video will download to your Desktop :D
</details>
